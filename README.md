# Graphic Designer Portfolio

Premium multi-page portfolio website built with HTML, CSS, and JavaScript.

## Pages
- Home
- About
- Portfolio
- Services
- Testimonials
- Contact

## Features
- Black and gold premium visual direction
- Sticky navigation across all pages
- Smooth animation reveals
- Loading screen animation
- Light and dark mode toggle with saved preference
- Portfolio filtering on the Portfolio page
- Contact form validation
- Responsive layout for mobile, tablet, and desktop

## Run locally
Open index.html directly in a browser, or run a local static server.

PowerShell example:
python -m http.server 5500

Then visit:
http://localhost:5500
